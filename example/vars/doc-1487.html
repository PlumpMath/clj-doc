while(clojure.core)

Macro

  • (while test & body)

Repeatedly executes body while test expression is true. Presumes some side-effect will cause test to become false/nil. Returns nil

; core.clj:3794
(defmacro while
  [test & body]
  `(loop []
     (when ~test
       ~@body
       (recur))))
