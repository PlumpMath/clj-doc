file(clojure.contrib.duck-streams)

  • (file & args)

Concatenates args as strings returns a java.io.File. Replaces all / and \ with File/separatorChar. Replaces ~ at the start of the path with the user.home system property.

; duck_streams.clj:53
(defn #^File file
  [& args]
  (let [#^String s (apply str args)
        s (.replace s \/ File/separatorChar)
        s (.replace s \\ File/separatorChar)
        s (if (.startsWith s "~")
            (str (System/getProperty "user.home")
                 File/separatorChar (subs s 1))
            s)]
    (File. s)))
