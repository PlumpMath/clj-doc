ensure(clojure.core)

  • (ensure ref)

Must be called in a transaction. Protects the ref from modification by other transactions. Returns the in-transaction-value of ref. Allows for more concurrency than (ref-set ref @ref)

; core.clj:1320
(defn ensure
  [#^clojure.lang.Ref ref]
    (. ref (touch))
    (. ref (get)))
