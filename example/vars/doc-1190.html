in-case(clojure.contrib.fcase)

Macro

  • (in-case test-value & clauses)

Like case, but test expressions are sequences. The test expression is true if any item in the sequence is equal (tested with '=') to the test value.

; fcase.clj:91
(defmacro in-case
  [test-value & clauses]
  `(fcase in-case-test ~test-value ~@clauses))
