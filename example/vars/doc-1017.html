slurp(clojure.core)

  • (slurp f)
  • (slurp f enc)

Reads the file named by f using the encoding enc into a string and returns it.

; clojure/core.clj:3182
(defn slurp
  ([f] (slurp f (.name (java.nio.charset.Charset/defaultCharset))))
  ([#^String f #^String enc]
  (with-open [r (new java.io.BufferedReader
                  (new java.io.InputStreamReader
                    (new java.io.FileInputStream f) enc))]
    (let [sb (new StringBuilder)]
      (loop [c (.read r)]
        (if (neg? c)
          (str sb)
          (do
            (.append sb (char c))
            (recur (.read r)))))))))
