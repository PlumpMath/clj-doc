aget(clojure.core)

  • (aget array idx)
  • (aget array idx & idxs)

Returns the value at the index/indices. Works on Java arrays of all types.

; core.clj:2172
(defn aget
  ([array idx]
   (clojure.lang.Reflector/prepRet (. Array (get array idx))))
  ([array idx & idxs]
   (apply aget (aget array idx) idxs)))
