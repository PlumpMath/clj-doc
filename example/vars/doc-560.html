aset(clojure.core)

  • (aset array idx val)
  • (aset array idx idx2 & idxv)

Sets the value at the index/indices. Works on Java arrays of reference types. Returns val.

; clojure/core.clj:2328
(defn aset
  ([array idx val]
   (. Array (set array idx val))
   val)
  ([array idx idx2 & idxv]
   (apply aset (aget array idx) idx2 idxv)))
