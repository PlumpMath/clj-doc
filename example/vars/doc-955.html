ref-max-history(clojure.core)

  • (ref-max-history ref)
  • (ref-max-history ref n)

Gets the max-history of a ref, or sets it and returns the ref

; clojure/core.clj:1463
(defn ref-max-history
  ([#^clojure.lang.Ref ref]
    (.getMaxHistory ref))
  ([#^clojure.lang.Ref ref n]
    (.setMaxHistory ref n)))
