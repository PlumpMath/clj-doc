nthrest(clojure.core)

  • (nthrest coll n)

Returns the nth rest of coll, (seq coll) when n is 0.

; core.clj:2426
(defn nthrest
  [coll n]
    (loop [n n xs (seq coll)]
      (if (and xs (pos? n))
        (recur (dec n) (rest xs))
        xs)))
