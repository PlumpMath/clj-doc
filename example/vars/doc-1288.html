partition-by(clojure.contrib.seq-utils)

  • (partition-by f coll)

Applies f to each value in coll, splitting it each time f returns a new value. Returns a lazy seq of lazy seqs.

; seq_utils.clj:73
(defn partition-by 
  [f coll]
  (when-let [s (seq coll)]
    (let [fv (f (first s))
          run (take-while #(= fv (f %)) s)]
      (lazy-cons run (partition-by f (drop (count run) s))))))
