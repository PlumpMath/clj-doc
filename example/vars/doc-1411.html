rseq(clojure.core)

  • (rseq rev)

Returns, in constant time, a seq of the items in rev (which can be a vector or sorted-map), in reverse order. If rev is empty returns nil

; core.clj:971
(defn rseq
  [#^clojure.lang.Reversible rev]
    (. rev (rseq)))
