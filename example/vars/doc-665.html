defunbound(clojure.contrib.def)

Macro

  • (defunbound name)
  • (defunbound name doc)

Defines an unbound var with optional doc string

; clojure/contrib/def.clj:32
(defmacro defunbound
  ([name]
     (list `def name))
  ([name doc]
     (list `def (with-meta name (assoc (meta name) :doc doc)))))
