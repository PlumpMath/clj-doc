postwalk(clojure.contrib.walk)

  • (postwalk f form)

Performs a depth-first, post-order traversal of form. Calls f on each sub-form, uses f's return value in place of the original. Recognizes all Clojure data structures except sorted-map-by. Consumes seqs as with doall.

; walk.clj:52
(defn postwalk
  [f form]
  (walk (partial postwalk f) f form))
