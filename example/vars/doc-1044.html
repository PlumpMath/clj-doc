symbol(clojure.core)

  • (symbol name)
  • (symbol ns name)

Returns a Symbol with the given namespace and name.

; clojure/core.clj:358
(defn symbol
  ([name] (if (symbol? name) name (clojure.lang.Symbol/intern name)))
  ([ns name] (clojure.lang.Symbol/intern ns name)))
