apply-template(clojure.contrib.template)

  • (apply-template expr values)

Replaces _1, _2, _3, etc. in expr with corresponding elements of values. Returns the modified expression. For use in macros.

; template.clj:128
(defn apply-template
  [expr values]
  (when-not (template? expr)
    (throw (IllegalArgumentException. (str (pr-str expr) " is not a valid template."))))
  (let [expr (postwalk-replace {'_ '_1} expr)
        holes (find-holes expr)
        smap (zipmap holes values)]
    (prewalk-replace smap expr)))
