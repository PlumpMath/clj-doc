take(clojure.core)

  • (take n coll)

Returns a lazy seq of the first n items in coll, or all items if there are fewer than n.

; core.clj:1464
(defn take
  [n coll]
    (when (and (pos? n) (seq coll))
      (lazy-cons (first coll) (when (> n 1) (take (dec n) (rest coll))))))
