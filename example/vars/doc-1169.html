frequencies(clojure.contrib.seq-utils)

  • (frequencies coll)

Returns a map from distinct items in coll to the number of times they appear.

; seq_utils.clj:83
(defn frequencies
  [coll]
  (reduce (fn [counts x]
              (assoc counts x (inc (get counts x 0))))
          {} coll))
