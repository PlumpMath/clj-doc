xml-zip(clojure.zip)

  • (xml-zip root)

Returns a zipper for xml elements (as from xml/parse), given a root element

; clojure/zip.clj:47
(defn xml-zip
  [root]
    (zipper (complement string?) 
            (comp seq :content)
            (fn [node children]
              (assoc node :content (and children (apply vector children))))
            root))
