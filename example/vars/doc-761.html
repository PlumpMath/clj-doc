ifn?(clojure.core)

  • (ifn? x)

Returns true if x implements IFn. Note that many data structures (e.g. sets and maps) implement IFn

; clojure/core.clj:4020
(defn ifn?
  [x] (instance? clojure.lang.IFn x))
