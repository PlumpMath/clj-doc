defhinted(clojure.contrib.def)

Macro

  • (defhinted sym init)

Defines a var with a type hint matching the class of the given init. Be careful about using any form of 'def' or 'binding' to a value of a different type. See http://paste.lisp.org/display/73344

; def.clj:79
(defmacro defhinted
  [sym init]
  `(do
     (def ~sym ~init)
     (alter-meta! (var ~sym) assoc :tag (class ~sym))
     (var ~sym)))
