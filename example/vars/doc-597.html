butlast(clojure.core)

  • (butlast coll)

Return a seq of all but the last item in coll, in linear time

; clojure/core.clj:190
(def 
 butlast (fn butlast [s]
           (loop [ret [] s s]
             (if (next s)
               (recur (conj ret (first s)) (next s))
               (seq ret)))))
