mapcat(clojure.core)

  • (mapcat f & colls)

Returns the result of applying concat to the result of applying map to f and colls. Thus function f should return a collection.

; clojure/core.clj:1632
(defn mapcat
  [f & colls]
    (apply concat (apply map f colls)))
