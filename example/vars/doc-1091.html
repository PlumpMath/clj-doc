defvar(clojure.contrib.def)

Macro

  • (defvar name)
  • (defvar name init)
  • (defvar name init doc)

Defines a var with an optional intializer and doc string

; def.clj:19
(defmacro defvar
  ([name]
     (list `def name))
  ([name init]
     (list `def name init))
  ([name init doc]
     (list `def (with-meta name (assoc (meta name) :doc doc)) init)))
