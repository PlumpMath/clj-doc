rename(clojure.set)

  • (rename xrel kmap)

Returns a rel of the maps in xrel with the keys in kmap renamed to the vals in kmap

; set.clj:47
(defn rename
  [xrel kmap]
    (set (map #(rename-keys % kmap) xrel)))
