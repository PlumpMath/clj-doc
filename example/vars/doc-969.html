rename-keys(clojure.set)

  • (rename-keys map kmap)

Returns the map with the keys in kmap renamed to the vals in kmap

; clojure/set.clj:70
(defn rename-keys
  [map kmap]
    (reduce 
     (fn [m [old new]]
       (if (not= old new)
         (-> m (assoc new (m old)) (dissoc old))
         m)) 
     map kmap))
