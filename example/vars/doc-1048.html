case(clojure.contrib.fcase)

Macro

  • (case test-value & clauses)

Like cond, but test-value is compared against the value of each test expression with =. If they are equal, executes the "body" expression. Optional last expression is executed if none of the test expressions match.

; fcase.clj:67
(defmacro case
  [test-value & clauses]
  `(fcase = ~test-value ~@clauses))
