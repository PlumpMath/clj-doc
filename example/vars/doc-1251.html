make-node(clojure.zip)

  • (make-node loc node children)

Returns a new branch node, given an existing node and new children. The loc is only used to supply the constructor.

; zip.clj:65
(defn make-node
  [loc node children]
    ((:zip/make-node ^loc) node children))
