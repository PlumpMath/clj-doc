get(clojure.core)

  • (get map key)
  • (get map key not-found)

Returns the value mapped to key, not-found or nil if key not present.

; core.clj:891
(defn get
  ([map key]
   (. clojure.lang.RT (get map key)))
  ([map key not-found]
   (. clojure.lang.RT (get map key not-found))))
