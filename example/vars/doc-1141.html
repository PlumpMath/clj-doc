every?(clojure.core)

  • (every? pred coll)

Returns true if (pred x) is logical true for every x in coll, else false.

; core.clj:1469
(defn every?
  [pred coll]
    (if (seq coll)
      (and (pred (first coll))
           (recur pred (next coll)))
      true))
